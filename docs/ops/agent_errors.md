# Agent Error Log Template

Use this template to capture every error encountered by any agent.

## Entry Template
- Context: phase, roadmap step, agent, commit SHA
- Symptom: concise description of the observed failure
- Root Cause: analysis of why the failure occurred
- Fix: remediation applied to resolve the issue
- Prevention: guard, documentation, or process update to avoid recurrence

## Notes
Every error entry must populate all fields and be linked to a roadmap step.
